Hong Kong Java User Group

The Hong Kong Java User Group holds occasional meetings, typically on Saturday afternoons. All meetings are free of charge. The official web site is at https://hkjug.dev.java.net/.

The group was selected by Sun Microsystems as one of their Top 25 Java User Groups worldwide.
