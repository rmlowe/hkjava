Example
Each PSM transformed into Code
•Code generated for the DBMS PSM
–SQL scripts to create and drop database tables
–SQL scripts to load initial data
•Code generated for the EJB PSM
–Home interfaces, Remote/Local Interface
–Entity beans and Session beans, Message driven beans
–Primary key classes and Finder classes
–Helper classes for update objects and data objects
–Deployment descriptors for the selected EJB server
•Code generated for the WEB PSM
–Action classes - perform various functions such as browse, create, and so on
–Form bean - set and get methods for the attributes submitted via the JSP
–JSP - generate the HTML output sent to the client
–Resource bundles - for internationalization of the Web application
