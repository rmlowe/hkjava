J2EE Developer Desires
Ÿ Data model should not constrain object model
Ÿ Don’t want database code in object/component code
Ÿ Accessing data should be fast
Ÿ Minimize calls to database – they are expensive
Ÿ Object-based queries – not SQL
Ÿ Isolate J2EE app from schema changes
Ÿ Notification of changes to data occurring at database
