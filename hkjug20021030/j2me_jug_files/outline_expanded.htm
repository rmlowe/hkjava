Set Up Development Environment
Install JDK
Install j2me_wireless_toolkit-1_0_3-win
Download Nokia Series60 j2me SDK
www.nokia.com -> developer -> java-> tools-> -> nS60_jme_sdk_b0_1a.zip
Install the above file, then copy the whole install directory to J2WTK/ wtklib/devices

Run the sample
Open a sample project included in J2WTK
Build
Project -> Package
Select device Nokia_Series_60
Run

J2me on 7650
It support J2me1.0
It has nokia UI API (non j2me standard)
It has sound API (non j2me standard)
3650 will have mobile media API to play music, video, and access the camera

7650-network
In 7650, u need to process the network process in a Thread other than main midlet
The html style return, can’t have the encoding in Http header as Big-5, GB3212
In 7650c, to read the chinese data from network, u need to read it as UTF

7650 – nokia UI
Full screen canvas
Direct graphics
Sound
LCD light control

7650c - bug
On 7650c, Form, Canvas can append chinese string
For control as List, Choice group, u need to code one using canvas
Can’t input chinese yet
