HKJava.com > Hong Kong Java User Group, 30 October 2002

Time: 19:00 (presentation starts at 19:30)
Venue: INETS, 2 Floor Chuang's Enterprise Building, 382 Lockhart Road, Wanchai
Topic: Developing for J2ME™ and the Nokia 7650
Speaker: Hugo Tung

Click here for Hugo's presentation

Hugo Tung will talk about programming for the Java™ 2 Platform, Micro Edition (J2ME™) in general, and the Nokia 7650 mobile phone in particular. This will be a very "hands-on" presentation; Hugo will be showing us lots of code examples using the J2ME and Nokia APIs.

Hugo is currently working on an interactive guide to Hong Kong for the Nokia 7650 using J2ME technology.
