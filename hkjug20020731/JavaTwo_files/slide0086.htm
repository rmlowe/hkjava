Server Side Development using Flash
Remoting
No additional effort is required for Java or ColdFusion Developers to
use Flash Remoting.
•  For EJB's, pass the JNDI name that the EJB was bound to in the
NetServices getService function.
•  For JMX MBeans, pass the JMX ObjectName to
the NetServices getService function.
•  JavaBeans are stateful to the client, Java classes are not.
