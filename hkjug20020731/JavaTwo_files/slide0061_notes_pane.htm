Salsa
Provides the server-side interface to popular web application servers including ColdFusion, JRun, .NET, and the leading Java servers.
Server-side developers write in their native languages (CFML, Java, or .NET)
The Flash Player communicates with the application server using the Action Message Format (AMF) over http.
AMF is a fast, lightweight binary format that allows the Flash Player to communicate with the application server
If you are thinking that your customers will wonder why we are using AMF instead of SOAP given that SOAP is the current industry standard for web data transfer…
AMF is a much fast format because it is binary.  You can think of it as a faster, binary version of SOAP.
SOAP processing was too slow to add to the Flash Player
Therefore, AMF support has been added to the Flash Player 6
