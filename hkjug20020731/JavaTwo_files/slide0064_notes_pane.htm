Salsa solves problems for both the Flash Developer and the Web Application Developer.
For the Flash Developer,
Salsa simplifies the process of connecting Flash to a web application server
Salsa also provides a rich debugging environment for the Flash Developer to debug applications
For the Web Application Developer,
Application developers will now be able to expose services to Flash using their existing programming environment.
Salsa simplifies how both input and output data is delivered to a Flash UI.
