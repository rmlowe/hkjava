Server Side Development using Flash Remoting

ActionScript Example - JavaBean
 // connect to Flash Remoting
gatewayConnnection = NetServices.createGatewayConnection();
 // get a reference to the remote JavaBean class
// "com.mypackage.FlashJavaBean"
 myJavaBean = gatewayConnnection.getService("com.mypackage.FlashJavaBean", this);
// invoke any methods available on the JavaBean class:
myJavaBean.setName( text );  // etc.
