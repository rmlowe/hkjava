While there are some great examples of Rich Internet Applications already, to date there has not been a complete solution for building these applications. Macromedia MX is the first product family to address this opportunity with all the technology that is required to be successful.

There are basically three elements to delivering rich internet applications:

– First is rich client technology that can provide a high-performance, small foot-print, cross-platform, and cross-device runtime environment for application clients.

CLICK 1 - Macromedia MX provides this with the Macromedia Flash Player. Flash Player extends what is possible with HTML in the browser and supports a wide range of devices. We also continue to strongly target HTML and XHTML solutions and all talk more about that. With more than 98% of web users already able to see Flash content and more than 2 million downloads a day, macromedia flash player is the standard for rich client technology.

– Second is a server scripting environment that is designed for integration with rich client technology, can support high-performance  connectivity, and work with leading Java and Microsoft application server technology.

CLICK 2 - Macromedia MX provides this with the new ColdFusion MX release, which offers that a rapid scripting environment that extends Java and .NET for building dynamic web sites and rich internet applications that offer exceptional user experiences.

– Third and most importantly, is a suite of powerful development tools that can enable the creation of these applications.

CLICK 3 - To enable developers and designers Macromedia is offering the essential suite of tools for building web sites and rich internet applications. Dreamweaver MX provides the environment for designing HTML pages, and coding server logic. Flash MX provides a powerful environment for building rich content and application user interfaces. Fireworks MX and FreeHand support developing graphics for content and applications. All of the tools are now being sold together in Macromedia Studio MX, and they’ve all been designed to work together as an integrated suite. They now share a common user interface standard that makes moving between the three products fast and easy. And they deliver powerful new features for developing everything form websites to rich internet applications.
