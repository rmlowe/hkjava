Sun Microsystems Inc. runs a program called Software Environment and Java Ambassador. Each year, as part of this program, there is a technology conference in the US; one of the purposes of the event is to provide an update on technology development to the ambassadors from different countries. At this meeting, Mickey Fan, Hong Kong's Java Ambassador, will talk about some of the most interesting topics discussed during the meeting.
